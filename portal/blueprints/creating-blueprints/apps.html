Apps in Sandboxes

The following actions are available for Apps in the sandbox:

📄️ View the Apps Catalog in the Blueprint

To view Apps:

📄️ Add an App to the Blueprint

This article explains how to add an App to your blueprint.

📄️ Edit an App's Details in the Blueprint

For Apps connected to a subnet, all Apps in the same subnet must use the same cloud provider resource.

📄️ Reset an App's Settings in the Blueprint (Refresh From Template)

Use this option to revert back to the definitions of the App template. Note that any changes made to the App itself in the blueprint will be lost.

This procedure explains how to delete Apps from a blueprint.
